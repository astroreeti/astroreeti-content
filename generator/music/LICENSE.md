# Music licensing

All background tracks in this folder are **CC0 1.0 (public domain)** — free for
commercial use, no attribution required, safe for Instagram Reels.

| File | Original artist | Source |
|------|-----------------|--------|
| floating-temple.mp3 | Monplaisir | CC0-1.0-Music corpus (chosic.com) |
| day-dreams.mp3 | HoliznaCC0 | Free Music Archive (CC0) |
| lucid-dreaming.mp3 | HoliznaCC0 | Free Music Archive (CC0) |

Tracks were trimmed to ~45s, loudness-normalized (-18 LUFS) and given
1.5s fades. Because they are CC0, there is zero copyright-strike risk when
published via the API (unlike Instagram's in-app licensed library, which the
publishing API cannot attach).
